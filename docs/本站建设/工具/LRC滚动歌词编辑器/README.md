## 使用说明

### 1. 选择本地音频文件
![](./img/img_2.png)
![](./img/img_3.png)

### 2. 粘贴歌词
:::tip 提示
从网上搜索相关歌词复制到歌词框，或者自己知道音频内容输入歌词。每一行为一个时间标记点，点击加入时间标记后光标会自动移到下一行首位。
:::
![](./img/img_4.png)

### 3. 开始标记

点击播放按钮开始播放歌曲
![](./img/img_5.png)

点击加入时间标记按钮添加时间标记

注意一开始的光标位置，应该手动点击到第一行最左边
![](./img/img_6.png)
